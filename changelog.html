  • Run the secured Spring MVC web application using an embedded Tomcat instance without the need of installing a local Tomcat Server

The template of the secured Spring MVC web application manage an authenticated user from administrator to generic user point of view. Supported features are:

Functional features for generic users: * Sig up, Sig in, Sig out on the web application * Chage user settings (e.g., first name, last name, password, …) * Chage user avatar

Functional features for administrator users (more than generic user): * Create, Update, Delete and Disable an user account * Reset an user password * Expire an user session
